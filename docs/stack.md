# Stack Decision

## Mobile
- Flutter for a single codebase targeting iOS, Google Play, and Huawei AppGallery.

## Backend
- Node.js + Express API (simple and fast to iterate).
- PostgreSQL planned for production; in-memory data for MVP scaffolding.
- Object storage (S3-compatible) for images.

## Admin
- Lightweight web dashboard (static HTML/JS) for moderation.

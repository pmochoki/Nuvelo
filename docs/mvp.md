# MVP Scope

## Must-have flows
- Browse and search listings
- Post a listing with photos
- Contact seller via in-app chat
- Manage profile and "My Listings"
- Report listings/users/messages
- Admin moderation of reports and listings

## Categories for launch
All categories are included at launch, with a unified listing model and category-specific fields.

## Exclusions (post-MVP)
- Paid promotions and subscriptions
- Advanced search relevance ranking
- Full identity verification and KYC
